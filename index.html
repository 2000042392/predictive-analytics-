Processing Pipeline

01 · INGEST
Load Dataset

CSV upload or built-in synthetic generator

02 · CLEAN
Preprocess

Dedup → impute → lag features → rolling mean

03 · SPLIT
Train / Test

TimeSeriesSplit cross-validation (5 folds)

04 · FIT
Train Model

Choose from 6 regression / time-series models

05 · EVALUATE
Score

MAE, RMSE, R², MAPE on held-out test set

06 · FORECAST
Predict Future

12-period ahead forecast with confidence band

Models

Six Forecasting Models

From interpretable baselines to powerful ensembles — compare every approach on the same dataset.

📈

Linear Regression

Baseline model. Fits a straight-line trend to time index and lag features. Fast and highly interpretable.

sklearninterpretablefast
〰️

Polynomial Regression

Captures non-linear trends using degree-3 features. Regularised with Ridge to prevent overfitting.

PolynomialFeaturesRidgenon-linear
🌳

Random Forest

Ensemble of 100 decision trees. Handles non-linearity and interactions between lag features naturally.

ensemblen=100robust
⚡

Gradient Boosting

Sequential boosting with gradient descent. Often the top performer on tabular time-series data.

GBMn=100accurate
📊

Moving Average

Classic time-series baseline. Averages the last 6 values. Zero training needed — pure heuristic.

window=6time-seriesno-train
🔭

Exponential Smoothing

Weights recent observations more heavily. Controls recency bias with α=0.3 smoothing factor.

α=0.3time-seriesweighted

Usage

Get Started in Minutes

Clone the repo, install dependencies, run the engine. No configuration files needed.

Installation

# Clone the repository git clone https://github.com/yourname/predictive-analytics cd predictive-analytics # Install dependencies pip install -r requirements.txt # Run with defaults (sales + gradient_boost) python3 src/predictive_model.py # Choose dataset and model python3 src/predictive_model.py users random_forest # Generate all charts python3 src/generate_charts.py

Use as a Module

from src.predictive_model import ( generate_dataset, preprocess, train_and_forecast ) # 1. Load data df = generate_dataset("sales") # 2. Preprocess df, log = preprocess(df) # 3. Train + forecast result = train_and_forecast(df, "gradient_boost") # 4. Inspect print(result["metrics"]) # {'mae': 3241.2, 'rmse': 4108.7, 'r2': 0.941} print(result["future_forecast"]) # [98234.1, 97881.4, 99012.6, ...]

Learning Outcomes

What This Project Teaches

🧹

Data Preprocessing

Deduplication, median imputation, lag feature engineering, and rolling statistics on real time-series data.

🔀

Train/Test Splitting

Why you can't use random split for time series. TimeSeriesSplit cross-validation prevents data leakage.

📐

Model Evaluation

MAE, RMSE, R², and MAPE — when each matters and how to choose the right metric for your problem.

🔮

Future Forecasting

How to iteratively feed predictions back as lag features to forecast multiple steps ahead.

📊

Visualisation

Publication-quality matplotlib charts: residual plots, multi-model comparison bars, confidence bands.

🏗️

Software Design

Modular, testable Python: each function has a single responsibility and returns JSON-serialisable data.
